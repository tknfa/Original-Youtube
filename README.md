# Classic YouTube Mode

Minimal Manifest V3 scaffold for a Chrome extension that restores a cleaner, classic YouTube browse experience.

## Load Unpacked

1. Open `chrome://extensions` in Chrome.
2. Enable **Developer mode**.
3. Click **Load unpacked** and select this folder.

Next steps: fill in `styles.css` and `content.js` to remove Shorts, Explore chips, and ad UI elements.

## Theme

Open the extension options page to switch between Light (classic) and Dark mode. The link color below videos remains blue in both modes.

## Logo

Place your classic logo image in `assets/gifs/` and set the filename in `assets/gifs/manifest.json` under the `logo` field.
